# abi

This folder contains ABI files for 
Praxis smart contracts.

## What is ABI?
ABI is how the frontend talks 
to the smart contracts on Base.

## Contents
- PraxisAgent.json - Agent ABI
- PraxisToken.json - Token ABI
- PraxisTreasury.json - Treasury ABI

See the main README for full project overview.
